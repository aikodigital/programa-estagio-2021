Inciar com:
# npm init -y

instalar as dependencias que vou precisar:

# npm install express mysql2 sequelize cors

Iniciar orm:
# npx sequelize-cli init

Criando a Migrate:

# npx sequelize-cli migration:generate --name create-paradas

Criar a Tabela:
# npx sequelize-cli db:migrate
